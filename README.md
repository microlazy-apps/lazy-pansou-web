# lazy-pansou-web

懒猫微服包装：[fish2018/pansou-web](https://github.com/fish2018/pansou-web) — 高性能网盘资源搜索（前后端集成版）。

聚合多个 Telegram 频道 + 80+ 搜索插件，自动按网盘类型分类，秒级响应。

## 安装

懒猫商店搜 "PanSou" 一键安装。

## 使用

装好访问 `https://pansou.{你的微服域名}`，输入关键词即可。

## 数据持久化

`/lzcapp/var/data/` —— 缓存（默认 60 分钟 TTL）+ 日志。
